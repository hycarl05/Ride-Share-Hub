Build a modern full-stack ride-hailing web application prototype called "Prebet UPSI".

Purpose:
This application is a safer alternative to the current Telegram-based student transport booking system used at Universiti Pendidikan Sultan Idris (UPSI). Instead of booking rides through public group chats, students can request rides securely through the application. Only registered and verified drivers are allowed to accept bookings.

Technology:
- React + Vite
- Tailwind CSS
- Node.js + Express
- SQLite (or Supabase if easier)
- Responsive design
- Clean modern UI inspired by Grab/Uber

There are three user roles:
1. Student
2. Driver
3. Admin

==================================
STUDENT FEATURES
==================================

- Login/Register
- Dashboard
- Book a ride
- Enter:
  - Pickup location
  - Destination
  - Notes
- Fare estimation
- Ride history
- Cancel booking
- View driver profile
- Driver verification badge
- Driver vehicle information
- Live ride status

Ride Status:
- Searching Driver
- Driver Accepted
- Driver Arriving
- Passenger Picked Up
- Completed
- Cancelled

==================================
DRIVER FEATURES
==================================

Driver registration includes:

- Full Name
- Student ID (optional if external driver)
- IC Number
- Phone Number
- Vehicle Type
- Vehicle Plate Number
- Driving License Number
- Profile Photo
- Vehicle Photo

Driver dashboard:

- Toggle Online/Offline
- Receive ride requests
- Accept/Reject ride
- Update ride status
- Today's earnings
- Trip history
- Driver rating

Only approved drivers can go online.

SAFETY FEATURES
==================================

✓ Driver verification badge
